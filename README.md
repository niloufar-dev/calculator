# 🧮 Glassmorphism Calculator

A modern and responsive calculator built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**.

This project features a custom-built calculation engine that **does not use `eval()`**. It supports multi-step mathematical expressions while correctly following operator precedence.

## 🧠 How It Works

Instead of using JavaScript's built-in `eval()` function, this calculator:

- Parses the mathematical expression into tokens.
- Evaluates multiplication (`×`), division (`÷`), and percentage (`%`) first.
- Then performs addition (`+`) and subtraction (`−`).
- Returns the final result rounded to **3 decimal places**.

### Example

Expression:

```text
25 + 10 × 4 − 12 ÷ 3
```

Result:

```text
61
```

Another example:

```text
5 + 4 × 3
```

Output:

```text
17
```

instead of

```text
27
```
